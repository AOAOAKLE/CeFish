这是一款集鱼类拍照、PIT ID识别、体长、体宽、眼宽、鳍长等14个体型特征以及体重信息收集和记录于一体的软件，名字叫测鱼宝。与传统的手工测量记录相比，测鱼宝不仅大大减少了人工劳动，而且保证了测量结果的准确性和可靠性。最新版的测鱼宝软件下载地址在：
https://doi.org/10.6084/m9.figshare.24278065.v2
